# Legacy Python Project

This directory contains the original Python implementation of the ERPcheck automation tools.
These files have been superseded by the Electron/TypeScript implementation in the root directory.

## Contents
- `erp_client.py`: Original ERP automation script.
- `gui.py`: Tkinter-based GUI.
- `*_client.py`: Various scrapers for Naver/Kakao.
- `debug_*`: Debugging artifacts from the Python version.

## Usage
These files are kept for reference logic only. To run the current application, use `npm run dev` in the root directory.
